# Customer Clustering with K-Means

Project clustering pelanggan menggunakan algoritma K-Means berdasarkan gross income dan rating pelanggan.

## Dataset
Dataset berisi data pelanggan supermarket seperti:
- Gross Income
- Rating pelanggan

## Tools & Libraries
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Streamlit

## Features
- Menampilkan dataset pelanggan
- Proses clustering menggunakan K-Means
- Visualisasi hasil clustering
- Deployment menggunakan Streamlit

## Cara Menjalankan

```bash
streamlit run app.py
